Button example:

```jsx
<Heading size="lg">Heading LG</Heading>
<Heading>Heading MD</Heading>
<Heading size="sm">Heading SM</Heading>
```

### Props

| Prop name |  Type  | Default |                     Description |
| --------- | :----: | ------: | ------------------------------: |
| `size`    | string |    `md` | can be set to: `sm`, `md`, `lg` |
